Project Overview

This web application uses a machine learning model trained on medical symptom data to predict five common diseases:

  • Flu
  • Asthma
  • Pneumonia
  • Anxiety
  • Depression

Upon prediction, the system also recommends a professional and medically reviewed treatment guide.

Objective

The primary goal of this project is to assist in the early diagnosis and preliminary treatment guidance for patients, especially in resource-limited environments, using AI.
